## File sorter and renamer  
  
This script sorts the files in the specified folder by type (Images, Documents) and renames them by date of change with an ordinal number.  
### Features  
* Sort files by extension:
* .jpg, .png, .gif → Images
* .pdf, .txt, .docx → Documents  
* Renaming files in the YYYY-MM-DD\_XXX.ext format  
* Automatically create folders if there are none.  
### How to use  
1. Download the code or clone the repository  
2. Install Python 3.x  
3. Paste the path to the folder to sort into the 'path` variable
4. Run the script
### Work example  
Source folder:
* photo1.jpg  
* doc1.pdf  
* photo2.png  
  
**After launch:**
Images/  
* 2025-08-14\_001.jpg  
* 2025-08-14\_002.png  
  
Documents/  
* 2025-08-14\_001.pdf

* Автоматическое создание папок, если их нет

1. Скачайте код или клонируйте репозиторий  
2. Установите Python 3.x
